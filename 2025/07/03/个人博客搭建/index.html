个人博客搭建

参考网址：

手把手教你从零开始搭建个人博客，20分钟上手_搭博客-CSDN博客

基于Hexo平台快速建站

搭建环境：nodejs、 npm 、git

安装cnmp

npm install -g cnpm –registry=https://registry.npmmirror.com

直接修改npm资源获取地址（推荐）

npm config set registry https://registry.npmmirror.com

安装初始化Hexo

将Hexo博客部署到GitHub
